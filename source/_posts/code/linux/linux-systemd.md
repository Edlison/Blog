# 介绍

systemctl是Systemd的一个工具，负责控制systemd系统和服务管理器。

systemd是一个系统管理守护进程、工具和库的集合。

# 指令

开机自动启动（在引导时激活服务并启动或禁用服务）

`systemctl enable xxx`

启动服务

`systemctl start xxx`

查看状态

`systemctl status xxx`
