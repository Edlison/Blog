# Pass

对中间选中的牌进行移动。

右手控制的牌堆不同。

# 古典移牌

将下方的牌拉到上面。

以此衍生出很多不同的Pass，本质就是利用动作掩饰Pass的过程。

## 注意

- 右手的提牌发力，才能看起来非常快

- 右手的遮挡

# 海曼移牌

控制上方的牌堆，把下面的牌堆压上来。

# Half Pass

下方的牌堆在上方的牌堆遮挡下，完成翻转。

# Egle Change

- 双翻
- 最上面的牌抽到最下面（大拇指中指控制整幅牌，无名指抽起第一张牌）
- 右手遮挡

# 思考

做动作的时候，漏出来的手的部分动作越小越好。

在遮挡下的动作达到一定熟练度后，一定是以最小的动作幅度完成所有操作。
