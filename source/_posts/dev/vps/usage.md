# top

```sh
top
```

## cmd

- P: rank by cpu usage
- T: rank by time
- M: rank by memory
- q: quit
- m: change view of memory
- t: change view of cpu
- W: write into ~/.toprc

# disk usage

```sh
du -h  # show current dir
du -h /usr/bin
df -h  # show disk usage
```
